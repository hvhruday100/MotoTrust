# MVP Scope

## Must Have

- Customer sign-in with Firebase.
- Customer motorcycle registration.
- Fixed-price service packages.
- Pickup and drop booking.
- Service order status tracking.
- Genuine parts usage record.
- Photo/video/CCTV proof metadata.
- Digital service history.
- Operations dashboard for service staff.

## Should Have

- Notification updates for booking, pickup, service start, proof ready, and delivery.
- Admin control of service packages and fixed prices.
- Basic service-center assignment.

## Later

- Flutter mobile app.
- Payment gateway.
- Cloudflare R2 media migration.
- Inventory management.
- Advanced mechanic scheduling.
- Customer subscription packages.
- Insurance and warranty integrations.
